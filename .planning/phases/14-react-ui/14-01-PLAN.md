  function detectTraject(student):
    - periode and leerjaar from student fields (String(), lowercase/trim)
    - BJ1 patterns: ['bj1', '1e jaar', 'jaar 1', 'leerjaar 1', 'bj 1', 'klas 1'] — matched via indexOf in periode
    - BJ2 patterns: ['bj2', '2e jaar', 'jaar 2', 'leerjaar 2', 'bj 2', 'klas 2'] — matched via indexOf in periode
    - Periode is leading; leerjaar only as fallback when periode matches nothing
    - If leerjaar === '1' → 'bj1'; leerjaar === '2' → 'bj2'
    - Fallback: console.warn + return 'bj2'

  function berekenStatus(student) — status return chain:
    - traject = detectTraject(student)  [NOTE: app.js has activeTraject toggle — React version always uses detectTraject]
    - p = berekenPrognose(student, traject)
    - ongeoorloofd = student.verzuim?.ongeoorloofd ?? 0
    - heeftScores = p.totaalVoldoendeOfHoger + p.totaalOnvoldoende > 0
    - if (!heeftScores)                → { kleur: 'grijs',  label: 'Onbekend' }
    - if (p.label === 'negatief')      → { kleur: 'rood',   label: 'Risico' }
    - if (p.label === 'neutraal')      → { kleur: 'oranje', label: 'Let op' }
    - if (ongeoorloofd > 600)          → { kleur: 'oranje', label: 'Verzuim' }
    - if (p.label === 'sbc')           → { kleur: 'blauw',  label: 'Profieljaar SBC' }
    - if (p.label === 'sbl')           → { kleur: 'groen',  label: 'Op koers' }
    - if (p.label === 'versneld_sbc')  → { kleur: 'blauw',  label: 'Versneld SBC' }
    - if (p.label === 'naar_bj2')      → { kleur: 'groen',  label: 'Op koers BJ2' }
    - fallback                         → { kleur: 'groen',  label: 'Op koers' }
