| Boundary | Description |
|----------|-------------|
| actiepunten form inputs → actiepuntenStore | Onderwerp (text), datum (date), status (select) are user-supplied in FeedbackActiepuntenSection form |
| notitie textarea → saveKlassen | Free-text memo entered by mentor, stored in encrypted store |
| SVG from buildSpiderSVG → dangerouslySetInnerHTML | SVG string from spider.ts utility injected into DOM |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-14-05-01 | Tampering | FeedbackActiepuntenSection — onderwerp input XSS | accept | Actiepunten displayed as JSX string children in the list (auto-escaped). Only risk is form input → dangerouslySetInnerHTML, which is not present here. |
| T-14-05-02 | Tampering | SpiderChartCard — SVG via dangerouslySetInnerHTML | mitigate | buildSpiderSVG input is student.deelgebiedScores keyed strings (score levels from PDF parser). spider.ts sanitizeCssVar() strips non-alphanum from CSS var names. Axis labels from DEELGEBIEDEN constant (not user input). Accepted per D-14-10 decision. |
| T-14-05-03 | Tampering | NotitiesTextarea — free text XSS | accept | Textarea value stored as plain string in student.notitie; displayed back in textarea value prop (no innerHTML). React auto-escapes the value attribute. |
| T-14-05-04 | Information Disclosure | NotitiesTextarea — localStorage migration | accept | Reads and cleans up legacy mentor notes from localStorage. Data moves to encrypted store. Reads only the app's own 'mentordashboard_notities' key — no cross-origin access. |
