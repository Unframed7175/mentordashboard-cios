## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-16-01 | Tampering | autoDetectKlas naam composition | mitigate | `.trim()` + 255-char cap: naam is sliced to 255 chars before passing to createKlas — prevents excessively long names from corrupting storage keys |
| T-16-02 | Spoofing | createKlas duplicate reuse | accept | Duplicate match is case-insensitive exact string; an attacker controlling a PDF header could at most reuse an existing class they already have access to — no privilege escalation |
| T-16-03 | Denial of Service | parseSinglePDF on first file throws | accept | Existing error handler in handlePDFs catches exceptions from parseSinglePDF; autoDetectKlas lets errors bubble so the same catch path handles them; skipped counter increments |
</threat_model>
