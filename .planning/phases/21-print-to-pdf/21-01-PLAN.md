    Append the following CSS block verbatim at the END of src/index.css. Do not modify any existing rules. The block must appear after the last existing line in the file.

    /* ==========================================================================
       Print / PDF Export (Phase 21 — EXP-01..04)
       ========================================================================== */

    @page {
      size: A4;
      margin: 2cm;
    }

    .print-header {
      display: none;
    }

    @media print {
      body > * {
        display: none;
      }

      .print-target {
        display: block !important;
        max-width: 100% !important;
        padding: 0 !important;
        margin: 0 !important;
      }

      .print-header {
        display: block;
        margin-bottom: 1.5rem;
        padding-bottom: 0.75rem;
        border-bottom: 1.5px solid #009FE3;
      }

      .no-print {
        display: none !important;
      }

      .detail-section {
        break-inside: avoid;
      }

      * {
        print-color-adjust: exact;
        -webkit-print-color-adjust: exact;
      }
    }

    Rationale for each rule (do not include in the file, for executor awareness only):
    - `body > * { display: none }` hides the Tauri shell (AppShell, nav sidebar, class tabs) — only .print-target is re-shown.
    - `.print-target { display: block !important }` overrides the body rule for the DetailWeergave wrapper.
    - `.print-header { display: block }` reveals the print-only header that is `display: none` in normal view.
    - `.no-print { display: none !important }` suppresses nav buttons and the Afdrukken button from print output.
    - `break-inside: avoid` on `.detail-section` prevents a section card from splitting across pages.
    - `print-color-adjust: exact` preserves RAG badge background colors (green/amber/red) which browsers suppress by default.
  </action>
