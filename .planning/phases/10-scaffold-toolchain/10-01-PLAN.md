## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-10-01 | Tampering | index.html overwrite by scaffold | mitigate | Back up to index.html.bak before running scaffold (Task 2 Step 1) |
| T-10-02 | Tampering | package.json overwrite removes xlsx CDN dep | mitigate | Read existing package.json before scaffold; restore xlsx tarball dep in Task 3 merge |
| T-10-03 | Tampering | scripts cycle: dev → tauri dev → beforeDevCommand → npm run dev → loop | mitigate | Use separate "vite-dev" script for beforeDevCommand; "dev" maps to tauri dev only |
| T-10-04 | Denial of Service | Port 1420 already in use when tauri dev runs | accept | strictPort: true in vite.config.ts (Plan 02) makes failure explicit; low risk in dev environment |
</threat_model>
