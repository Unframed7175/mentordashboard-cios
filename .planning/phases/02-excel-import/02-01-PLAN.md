### 1. window.parseVerzuimTime(str)

Converts Dutch time format to total minutes. Addresses review concern [MEDIUM] about edge cases.

- Input: string like "107u24m", "0u45m", "12m", "2u05m", or null/undefined/empty
- Regex: `/^(?:(\d+)u)?(\d+)m$/i` — BOTH parts optional (hours OR minutes can appear alone)
- Also handle "HH:MM" format: `/^(\d+):(\d+)/`
- Also handle plain integers: `parseInt(s, 10)` if it's a bare number
- If input is already a number type, return `Math.round(str)`
- Edge cases that MUST work:
  - `"107u24m"` -> 6444
  - `"0u45m"` -> 45
  - `"12m"` -> 12 (no hours part)
  - `"2u05m"` -> 125
  - `"0u00m"` -> 0
  - `null` -> 0
  - `undefined` -> 0
  - `""` -> 0
  - `"  "` -> 0

Implementation:
```javascript
window.parseVerzuimTime = function(str) {
  if (str === null || str === undefined || str === '') return 0;
  if (typeof str === 'number') return Math.round(str);
  var s = String(str).trim();
  if (!s) return 0;
  // "107u24m", "0u45m", "12m", "3u"
  var m1 = s.match(/^(?:(\d+)u)?(\d+)m$/i);
  if (m1) return (m1[1] ? parseInt(m1[1], 10) : 0) * 60 + parseInt(m1[2], 10);
  // Handle "3u" without minutes
  var m1b = s.match(/^(\d+)u$/i);
  if (m1b) return parseInt(m1b[1], 10) * 60;
  // "HH:MM"
  var m2 = s.match(/^(\d+):(\d+)/);
  if (m2) return parseInt(m2[1], 10) * 60 + parseInt(m2[2], 10);
  // Plain integer
  var n = parseInt(s, 10);
  if (!isNaN(n)) return n;
  return 0;
};
```

**Strategy 2: normalizeNaam full match**
```javascript
var normV = window.normalizeNaam(v.naam);
student = window.appState.students.find(function(s) {
  return window.normalizeNaam(s.naam) === normV;

```

**Strategy 3: achternaam-only exact match**
- Extract achternaam from both sides: everything before the first comma (or full string if no comma)
- PDF format: "Achternaam, Initialen (Voornaam)" -> achternaam = part before comma
- Excel format: "Achternaam, Roepnaam" -> achternaam = part before comma
- Only attempt if achternaam length >= 3 (avoid false positives on short names)
```javascript
var excelAchternaam = window.normalizeNaam(v.naam).split(',')[0].trim();
if (excelAchternaam.length >= 3) {
  student = window.appState.students.find(function(s) {
    var pdfAchternaam = window.normalizeNaam(s.naam).split(',')[0].trim();
    return pdfAchternaam.length >= 3 && pdfAchternaam === excelAchternaam;
  });
}
```

**Strategy 4: PDF achternaam substring of Excel naam**
- For cases where name formats differ significantly
- Check if the PDF achternaam (>= 4 chars) appears anywhere in the normalized Excel naam
```javascript
var normV4 = window.normalizeNaam(v.naam);
student = window.appState.students.find(function(s) {
  var pdfAchternaam = window.normalizeNaam(s.naam).split(',')[0].trim();
  return pdfAchternaam.length >= 4 && normV4.indexOf(pdfAchternaam) !== -1;
});
```
